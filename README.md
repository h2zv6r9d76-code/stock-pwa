# 在庫管理PWA Ver.3

## 主な変更
- 一覧のスワイプ処理を全面的に作り直し
- 短い左スワイプ：赤い「削除」ボタンを表示
- 大きな左スワイプ：そのままゴミ箱へ移動
- 削除後5秒間「元に戻す」を表示
- ゴミ箱から復元・完全削除・一括削除
- 写真ライブラリ／カメラ／ファイル選択対応
- 既存のIndexedDBをそのまま利用

## 更新方法
このフォルダ内の8ファイルをGitHubの `stock-pwa` リポジトリへ上書きしてCommitしてください。
GitHub Pagesの設定変更は不要です。

## 更新後
GitHub Pagesの反映後、Safariで公開URLを再読み込みしてください。
古い表示が残る場合はホーム画面アプリを終了し、Safariで再読み込みしてから再度開いてください。
