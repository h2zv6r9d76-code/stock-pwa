# 在庫管理PWA Ver.2.2

## 追加機能
- 在庫を左へスワイプすると「削除」を表示
- 削除すると即時消去せずゴミ箱へ移動
- ゴミ箱から復元
- ゴミ箱から1件ずつ完全削除
- ゴミ箱を一括で空にする
- ゴミ箱件数を設定画面に表示

## データ互換性
Ver.2.1と同じIndexedDBを使用します。同じURLへ上書き更新する限り、既存在庫・画像・バックアップ互換性を維持します。

## 更新方法
このフォルダ内の8ファイルをGitHubの `stock-pwa` リポジトリへすべて上書きアップロードし、Commitしてください。
GitHub Pages設定の変更は不要です。
