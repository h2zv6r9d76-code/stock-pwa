# 在庫管理PWA Ver.3.1

既存の在庫データを保ったまま更新できる、iPhone向けのオフライン在庫管理PWAです。

## Ver.3.1 の追加内容

- 関連ワード欄を追加。商品名・分類・保管場所・メモと合わせて検索できます。
- 商品名に応じて、関連ワード候補を表示。候補を押すだけで追加できます。
- 以前使った分類・保管場所を候補として表示。入力を繰り返す必要がありません。
- Safariで安定しない「保存を保護する」操作を削除し、保存状況とバックアップの注意を表示します。

## 更新方法

このフォルダ内の全ファイルを、GitHub の `stock-pwa` リポジトリへ同名で上書きアップロードして `Commit changes` を押してください。GitHub Pages の設定変更は不要です。

更新前にアプリの「バックアップを書き出す」から、念のためバックアップを1つ保存してください。
